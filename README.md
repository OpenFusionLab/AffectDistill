# AffectDistill

> Official PyTorch implementation of **AffectDistill: On-Policy Distillation of Affective Reasoning from Multimodal Large Language Models for Multimodal Sentiment Analysis**.

## 🚀 Overview

AffectDistill is a lightweight multimodal sentiment analysis framework that distills affective reasoning from Multimodal Large Language Models (MLLMs) via on-policy knowledge distillation.

## ✨ Highlights

- On-policy affective reasoning distillation
- Prompt-aware teacher network
- Lightweight student model
- Efficient inference without MLLMs
